Crie um componente que possua um botão criar relatório que quando acionado deve mostrar um relatório de resultados da disciplina javascript para 20 alunos, no formato de tabela. As notas deverão ser números inteiros entre 0 e 100, criadas aleatoriamente. Cada aluno deve ser representado por um nome, número e nota. A impressão do relatório deve ter: um título, resultados mostrados linha a linha no formato " Aluno nr xx - Nota yy [(A/RE) PROVADO]"; e um rodapé com estatística final no formato "APROVADOS: XX (xx%) | REPROVADOS: YY (yy%)". Cada linha do relatório deve ser zebrada.
